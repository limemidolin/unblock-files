# unblock-files
Unblocks the files that are blocked by Windows.

## How to use
1. Download `Unblock.bat`
2. Move or Copy `Unblock.bat` to folder of contains blocked files that are blocked by Windows.
3. Execute it.
4. Confirm to files is not blocked.


## Requirements
* Windows PowerShell: >= 3.0
  * In this file, using `Unblock-File` Cmdlet. And unblock to recursive.
  * Later the Windows 8, PS 3.0 is Preinstalled. otherwise, please get from Microsoft download page.
